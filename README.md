# مركز استخبارات الطيران والسفر العالمي ✈️

## دليل النشر على GitHub Pages خطوة بخطوة

---

## الخطوة 1 — إنشاء حساب GitHub (مجاني)

1. افتح [github.com](https://github.com)
2. اضغط **Sign up**
3. أدخل بريدك الإلكتروني واختر اسم مستخدم وكلمة مرور
4. أكّد بريدك الإلكتروني

---

## الخطوة 2 — إنشاء مستودع جديد

1. بعد تسجيل الدخول، اضغط على **+** في الأعلى ثم **New repository**
2. في خانة **Repository name** اكتب: `aviation-hub`
3. اختر **Public**
4. ✅ ضع علامة على **Add a README file**
5. اضغط **Create repository**

---

## الخطوة 3 — رفع ملف index.html

1. داخل المستودع، اضغط **Add file** ثم **Upload files**
2. اسحب ملف `index.html` إلى الصفحة أو اضغط **choose your files**
3. في أسفل الصفحة اضغط **Commit changes**

---

## الخطوة 4 — تفعيل GitHub Pages

1. اضغط على **Settings** في المستودع
2. من القائمة الجانبية اختر **Pages**
3. تحت **Branch** اختر **main** ثم اضغط **Save**
4. انتظر دقيقة واحدة ثم أعد تحميل الصفحة
5. ستجد رابط موقعك مثل:
   ```
   https://yourusername.github.io/aviation-hub
   ```
